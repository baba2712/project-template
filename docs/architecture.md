# Architecture

This repository is a starter template for projects that want consistent AI-agent guidance, documentation, and automation from day one.

## Layout
- `.github/` contains Copilot instructions, skills, workflows, and contribution guidance.
- `src/` contains application or library code.
- `tests/` contains automated tests.
- `docs/adr/` contains architecture decision records.

## Design goals
- Minimal but extensible starting point.
- Strong defaults for documentation and testing.
- Clear guidance for both humans and AI agents.
